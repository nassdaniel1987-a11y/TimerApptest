# Abholzeiten Manager - Neumorphism UI/UX Design

## Übersicht

Alternatives **Neumorphism**-Design für die TimerApp, wählbar in den Einstellungen unter "Design" (Klassisch / Neumorphism).

- **Design-Stil:** Neumorphism - weiche, erhabene Elemente mit subtilen Schatten
- **Hintergrund:** Light Gray (#E0E5EC)
- **Akzentfarbe:** Soft Blue-Teal
- **Typografie:** Manrope (Headings), Inter (Body)
- **Sprache:** Deutsch
- **Plattform:** Android (360x800dp)

### Neumorphism Shadow Styles

**Raised (Buttons, Cards):**
```
shadow: 6px 6px 12px #A3B1C6, -6px -6px 12px #FFFFFF;
border-radius: 16px;
```

**Pressed/Inset (Active States, Input Fields):**
```
shadow: inset 4px 4px 8px #A3B1C6, inset -4px -4px 8px #FFFFFF;
border-radius: 12px;
```

**Flat (Hover States):**
```
shadow: 3px 3px 6px #A3B1C6, -3px -3px 6px #FFFFFF;
```

### Compose-Umsetzung (Referenz)

```kotlin
// Neumorphic Modifier Extension
fun Modifier.neumorphicRaised() = this
    .shadow(elevation = 6.dp, shape = RoundedCornerShape(16.dp))
    .background(
        color = Color(0xFFE0E5EC),
        shape = RoundedCornerShape(16.dp)
    )

fun Modifier.neumorphicPressed() = this
    .background(
        color = Color(0xFFE0E5EC),
        shape = RoundedCornerShape(12.dp)
    )
    // Inner shadow via custom draw
```

